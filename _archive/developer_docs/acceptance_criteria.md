Acceptance Criteria (MVP)

Providers & Import
- Gmail and Outlook connect via OAuth PKCE; scopes presented; tokens stored locally encrypted.
- Import last 90 days with progress, errors, pause/resume; dedupe by message IDs.

Analyses
- Ikigai and Marketing analyses run locally; deterministic seed; model/version visible; JSON outputs with schema_version, generator, coverage, confidence.

Recommendations
- ASIN/Switched/Amadeus produce ≥20 items with images and rationales based on derived categories/themes; user can save/hide.

Sharing
- Encrypted file export opens offline with embedded viewer; encrypted URL works for payload ≤100KB; Drive/OneDrive save and share link creation works.

Privacy & Security
- Redaction defaults ON for share; pre-share risky content check; Request Inspector shows no raw content leaves device.

Quality Gates
- Accessibility WCAG 2.2 AA; performance budgets met; zero high-severity security findings in baseline scans.
