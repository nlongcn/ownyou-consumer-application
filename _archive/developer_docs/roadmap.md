Roadmap

Phase 1 — Foundations
- PWA shell, crypto/lock screen, encrypted stores, OAuth PKCE for Gmail/Outlook, parsers in WebWorkers.

Phase 2 — Analyses
- Port analyzers to workers/WASM; deterministic seed; redaction; dashboards (Ikigai/Marketing) with coverage/confidence.

Phase 3 — Recommendations
- ASIN/Switched/Amadeus integrations (developer keys); rationale UI; filters and dedupe; safety.

Phase 4 — Sharing
- Encrypted file/URL exports; Drive/OneDrive connectors; public preview shell (client-side only).

Phase 5 — Quality
- A11y, performance, i18n, docs, threat model; automate tests and CI gate.
